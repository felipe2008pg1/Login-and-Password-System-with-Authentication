# LOGIN AND PASSWORD SYSTEM WITH AUTHENTICATION | 🇺🇸 English
### 📝 Description
- This is a backend authentication system developed in Python, focused on user management, credential validation, and data persistence. The project simulates a real-world application workflow, including protection against brute-force attacks (attempt limit) and external file handling.

### 🛠️ Features
- [User Registration]: Validation to prevent duplicate names in the system.
- [Login System]: Credential verification with database search.
- [Data Persistence (JSON)]: Automatic user storage in a .json file, ensuring data is not lost when the program is closed.
- [Basic Security]: Temporary lockout after 3 incorrect login attempts.
- [Utilities]: Current date/time display and database reporting for testing.

_Terminal Interface: Clean system with dynamic screen clearing functions (Windows and Linux compatible)._

### 🚀 Technologies
- Python 3.x
- json library (Data manipulation)
- os and sys libraries (Operating system integration)
